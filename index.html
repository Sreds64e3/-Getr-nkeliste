Neuen Verbrauch erfassen

In diesem Bereich können Sie den Getränkeverbrauch für einen Spieler an einem bestimmten Tag eintragen. Wählen Sie einfach den Spieler und das Datum aus und geben Sie die Anzahl der konsumierten Getränke in die entsprechenden Felder ein. Die Gesamtsumme wird automatisch berechnet. Mit einem Klick auf "Eintrag hinzufügen" wird der Verbrauch gespeichert und erscheint in der Liste "Alle Einträge".

Alle erfassten Einträge

Hier finden Sie eine vollständige Liste aller erfassten Getränkeverbräuche. Sie können die Liste nach Spieler oder Zahlungsstatus filtern, um schnell die gewünschten Informationen zu finden. Jeder Eintrag zeigt Details zum Verbrauch und den Gesamtkosten. Mit dem Schalter in der letzten Spalte können Sie einen Eintrag als "bezahlt" oder "offen" markieren. Fehlerhafte Einträge können über den "Stornieren" Button gelöscht werden.
